# Project
##  Описание проекта
Проект посвящён анализу пользовательских отзывов на тушь для ресниц бренда PUSY.  
Данные собраны с трёх крупнейших маркетплейсов:  
- Ozon  
- Wildberries  
- Золотое Яблоко


**Цель проекта** — с помощью методов обработки естественного языка (NLP) выявить:
- ключевые темы отзывов
- тональность (позитивную / негативную / нейтральную)
- основные проблемы продукта
- сильные стороны, важные для пользователей


##  Данные

- **Общий объём выборки:** 941 отзыв  
- **Распределение по источникам:**

| Платформа | Количество |
|-----------|------------|
| Ozon | 468 |
| Wildberries | 341 |
| Золотое Яблоко | 132 |

##  Этапы работы

### 1. Сбор данных
- Отзывы собирались вручную через сохранение HTML-страниц
- Использовался парсинг с BeautifulSoup
- Извлекались: текст отзыва, дата, источник

### 2. Предобработка
- Приведение дат к единому формату
- Удаление маркеров: «Достоинства», «Недостатки», «Комментарий»
- Лемматизация (pymorphy3)
- Удаление стоп-слов, эмодзи, специальных символов

### 3. Выделение ключевых слов
- Использовался метод TF-IDF
- Для каждого отзыва выделено топ-5 ключевых слов

**Самые частотные ключевые слова:**
 хороший, ресница, понравиться, цвет, отличный, удлинять, коричневый, красивый, осыпаться

### 4. Тематическое моделирование
- Использовалась библиотека BERTopic
- Выделено 8 устойчивых тем

| Тема | Описание |
|------|----------|
| Благодарности | Подарки, пробники, спасибо |
| Цвет | Оттенки: коричневый, чёрный |
| Смывание | Легко, вода, тёплый |
| Аллергия | Глаза, чувствительность |
| Комки | Текстура, ложится ровно / не ровно |
| Прокрашивание | Качество, плохо / хорошо |
| Эстетика | Красивый цвет, дизайн |
| Объём / удлинение | Функциональные ожидания |

### 5. Анализ тональности
- Модель: `cointegrated/rubert-tiny-sentiment-balanced`
- Классы: positive / neutral / negative

**Результаты:**
- Позитивные: **46,9%**
- Нейтральные: **31,7%**
- Негативные: **21,5%**

**Средняя тональность по площадкам:**
- Ozon: **0.29**
- Wildberries: **0.205**
- Золотое Яблоко: **0.192**


## Визуализации

В проекте построены:
- Распределение тем (барчарт)
- Intertopic Distance Map (BERTopic)
- Древовидная кластеризация тем
- Распределение тональности по отзывам
- Топ-10 позитивных и негативных слов

##  Основные выводы

### Сильные стороны продукта
- Удлинение и объём
- Хороший цвет
- Лёгкое смывание
- Положительная реакция на пробники и подарки

### Проблемные зоны
- Образование комочков
- Неравномерное прокрашивание
- Аллергические реакции у чувствительных пользователей

### Пользовательские сегменты
Потребители разделились на три сегмента: ценящие внешний вид и цвет, ориентированные на рабочие характеристики, и чувствительные к недостаткам продукта.
