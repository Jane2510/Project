# Project
# Анализ отзывов на тушь для ресниц бренда PUSY

##  1. Введение

В условиях высокой конкуренции на рынке косметической продукции анализ пользовательских отзывов становится важным инструментом для оценки качества товаров и восприятия бренда. Отзывы содержат ценные данные о реальном опыте потребителей, включая как положительные, так и отрицательные аспекты использования продукта.

Современные методы обработки естественного языка (NLP) позволяют автоматизировать анализ больших объемов текстовых данных, выявлять скрытые закономерности, определять эмоциональную окраску отзывов и выделять ключевые темы.

**Цель проекта** — анализ отзывов на тушь для ресниц бренда PUSY, собранных с различных онлайн-площадок, с использованием методов NLP.

**Задачи:**
- Сбор данных с маркетплейсов
- Предобработка и очистка текстов
- Анализ тональности
- Выделение ключевых слов
- Тематическое моделирование
- Сравнительный анализ проблем по площадкам

##  2. Описание компании

**PUSY** — это бренд косметической продукции, ориентированный на выпуск декоративной косметики для широкой аудитории потребителей. Основной акцент компании делается на сочетании доступной цены, современного дизайна и соответствия актуальным трендам индустрии красоты.

Бренд активно продвигается через онлайн-каналы, включая социальные сети и маркетплейсы, что делает отзывы пользователей одним из ключевых источников информации о восприятии продукции. Благодаря этому компания может оперативно отслеживать мнение потребителей и реагировать на выявленные недостатки.

### Продукт: тушь для ресниц PUSY

Тушь для ресниц PUSY относится к категории декоративной косметики и предназначена для:

| Функция | Описание |
|---------|----------|
| 💎 Увеличение объема | Придает ресницам визуальную густоту |
| 📏 Удлинение | Визуально удлиняет ресницы |
| 👁 Придание выразительности | Делает взгляд более открытым и ярким |

### Ключевые аспекты пользовательского опыта

Отзывы о туши содержат информацию о:

- Удобстве нанесения
- Стойкости
- Наличии или отсутствии комочков
- Реакции глаз (например, аллергия)
- Внешнем эффекте (объем, длина, цвет)


## 🛠 4. Используемые технологии

- **Python 3.x**
- **Pandas** — обработка данных
- **NLTK** + **pymorphy3** — токенизация и лемматизация
- **Scikit-learn** — TF-IDF для ключевых слов
- **BERTopic** — тематическое моделирование
- **Transformers (rubert-tiny)** — анализ тональности
- **KeyBERT** — выделение ключевых слов
- **Matplotlib** — визуализация
